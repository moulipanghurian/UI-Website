# UI-Website
Redesign Website

Saya mengonfirmasi bahwa desain UI yang saya buat ini adalah salah satu proyek yang pernah saya kerjakan sebagai bagian dari penelitian dan laporan akhir saya. Saya memiliki alasan kuat untuk meredesain website ini karena saya pernah magang di Dinas Kesehatan Provinsi Jawa Timur. Desain ini bukan desain resmi dan tidak dimaksudkan untuk menggantikan situs web resmi yang ada. Semua hak cipta terkait situs web resmi adalah milik Dinas Kesehatan Provinsi Jawa Timur. Tujuan dari publikasi ini adalah untuk mendokumentasikan penelitian dan hasil kerja saya, serta untuk memberikan kontribusi yang bermanfaat berdasarkan pengalaman praktis saya selama magang. Saya berharap hasil ini dapat memberikan masukan yang konstruktif bagi pengembangan website Dinas Kesehatan Provinsi Jawa Timur di masa depan.
